git init   -создаем git репозиторий из текушей папкт
git add .   -просто пишем эту команду каждый раз перед commit
git commit -m master "message"  - создаем новую версию
git push origin master  -закидываем версию в github
git pull origin master - стягиваем новые изменения с github (если они есть в github, но их нет локально)
git clone ssh-ссылка    - копируем репозитроий с github (можно чужой)
git log   - посмотреть все коммиты
